Front-End

JavaScript é essencial para interatividade. Uso em projetos com manipulação do DOM, eventos e consumo de APIs.

Node.js me permite criar servidores com JavaScript. Já desenvolvi APIs simples e aplicações com Express.

Visual Studio Code é meu editor principal. Uso extensões, atalhos e depuração para aumentar minha produtividade.

npm é o gerenciador de pacotes do Node. Uso para instalar dependências e automatizar tarefas com scripts.
